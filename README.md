# Fake-social-media-accounts-and-their-detection

## Project Title Finalization

The project title "Fake Social Media Accounts and Their Detection" was finalized. The project focuses on identifying fake social media accounts using account information and activity patterns.
---
## Requirement Gathering

# Problem Statement

Fake social media accounts are often used for spreading misinformation, spam, scams, and other malicious activities. Detecting these accounts is important for maintaining security and trust on social media platforms.
---
## Objective Definition

#Objectives

- Identify fake social media accounts.
- Analyze account profiles and activities.
- Detect suspicious behavior patterns.
- Improve platform security and user trust.
---
## User & Module Identification

#Modules

- User Authentication Module
- Account Data Management Module
- Fake Account Detection Module
- Dashboard Module
- Report Generation Module
- Admin Module
---
## Use Case Diagram Preparation

A use case diagram was created to represent the interaction between users and the system. It includes functionalities such as login, account analysis, fake account detection, and report generation.
---
## Database Requirement Analysis

The database requirements were analyzed to store user details, account information, detection results, and reports.

#Tables

- Users
- SocialMediaAccounts
- DetectionResults
- Reports
- Admin
---
## ER Diagram Design

An Entity Relationship Diagram was designed to define the relationships between the database tables and provide a clear database structure.
---
## Database Schema Creation

The database schema was created based on the ER Diagram. Tables, attributes, primary keys, and foreign keys were defined for efficient data management.
---
##UI Wireframe Design

Wireframes were designed for the application's main pages, including the Login Page, Dashboard, Account Analysis Page, and Result Page.
---
## Login & Dashboard UI Design

The Login Page and Dashboard UI screens were designed to provide a simple and user-friendly interface for users.
---
## Navigation & Form Design

Navigation flow between pages was designed, and forms were created for login, account data submission, and account analysis.
---
##Design Review

The project design, database structure, wireframes, and UI screens were reviewed and approved for implementation.
